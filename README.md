# Chocobot

## The Chocolate Edition Discord server automod bot

to set up:

duplicate `.env.example` and rename to `.env`
include the client secret

```powershell
python -m venv .venv
```

```powershell
.venv/Scripts/activate
```

```powershell
pip install -r requirements.txt
```

```powershell
pip install -e .
```

with activated terminal: run `chocobot`
